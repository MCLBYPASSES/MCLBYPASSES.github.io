The IRL BYPASS Guide

Note: Aimlock is extremely difficult to implement, so this guide will focus on using ESP (Extra Sensory Perception) instead.

Requirements

  • MYST: The external tool created by my friend.
  • Video Editing Software: Use Adobe Premiere Pro or any similar application that supports chroma keying.

Setup Instructions

Step 1: Obtain MYST

  • Download MYST and install it on your system.
  • For stealth purposes, rename it to a system file (e.g., system32) to avoid detection in Task Manager.

Step 2: Configure MYST

  • MYST can be loaded prior to launching Roblox.
  • Utilize the auto-configuration feature for automatic stream-proofing.
  • MYST is designed to hide from Task View and Taskbar. You can interact with MYST without changing your cursor appearance during recordings.

Step 3: Adjust Windows Defender

To avoid detection by Windows Defender, you can disable it or add an exclusion. Here’s how to do it without showing up in Windows Antivirus:

  1. Open Registry Editor:
    • Type regedit in the Windows search bar and hit Enter.
  2. Navigate to the Following Directory: 
    Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender
  3. Add a New Key:
    • Right-click on the right pane and select New > String Value.
    • Name it AddExclusion.
    • Set the value to the directory path of your MYST mapper.
    Note: MYST is not classified as a virus, so it should be safe to add.
  4. Permissions:
    • If you encounter issues editing the registry due to permission problems, search for a YouTube video tutorial on how to gain the necessary permissions. This may take around 10 minutes.

Step 4: Start Your Recording

  • You’re now ready to record! MYST is configured, and Windows Defender is bypassed. Both Task Manager and Task View will not show any suspicious activity.

Step 5: Using ESP During Recording

  • Start your IRL recording and utilize your ESP tool as needed.

Post-Recording Process

  1. Open Premiere Pro.
  2. Get the Hex Color of Your ESP:
    • This information will be provided within MYST.
  3. Remove the ESP Color:
    • Input the hex color into Premiere Pro’s chroma key settings to remove that color from your video.
    • Important: Choose a hex color that is not present in your environment or the game (commonly used colors are green or purplish).

With these steps, you should be able to effectively use MYST and record your sessions without detection! Happy streaming!
